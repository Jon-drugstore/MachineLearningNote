## General Debug Options
----
想提高预测精度，往往采用：

|            手段           |            评价              |
| ------------------------ | --------------------------- |
| 采集更多样本               | 见多识广会让人变聪明，但也聪明反被聪明误。 |
| 降低特征维度               | 被降的维度也许有用。 |
| 采集更多特征               | 增加计算负担，也可能导致过拟合。 |
| 高次多项式回归              | 可能过拟合。 |
| 调试正规化参数$$\lambda$$  | 调节策略缺乏指导，只是猜测性调节。 |
